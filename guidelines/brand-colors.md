# KOMISSIO — Color System

> Purple Era — v2 (2024)

## Primary Violet Ramp

| Name | Hex | Usage |
|------|-----|-------|
| Void | `#09061A` | Hero and dark section backgrounds |
| Dark | `#150D2E` | Dark card / panel backgrounds |
| Deep | `#4C1D95` | Bottom-left mark block; deep UI accents |
| Violet | `#6D28D9` | **Primary brand colour** — mark top bar, CTAs, links |
| Mid | `#7C3AED` | Interactive hover states, icon tile gradient |
| Vivid | `#8B5CF6` | Charts, highlights, selected states |
| Light | `#A78BFA` | Bottom-right mark block; split accent |

## Surface & Neutral

| Name | Hex | Usage |
|------|-----|-------|
| Pale | `#C4B5FD` | Surface borders, mono mark mid-block |
| Surface | `#EDE9FE` | Card backgrounds in light mode |
| Off-white | `#F5F3FF` | Page background (default light mode) |
| White | `#FDFCFF` | Pure white panels, max contrast |
| Ink | `#1E1B4B` | Body text, headings |
| Muted | `#6B6785` | Labels, captions, secondary copy |

## Usage Rules

- **Primary mark on light backgrounds**: use `#6D28D9` (Violet) top bar, `#4C1D95` (Deep) bottom-left, `#A78BFA` (Light) bottom-right.
- **Reversed mark on dark backgrounds**: use white at full / 0.46 / 0.88 opacity for the three blocks.
- **Monochrome contexts**: use `#6D28D9` at full / 0.38 / 0.72 opacity.
- Never place the primary mark on Pale (`#C4B5FD`) or lighter violet surfaces — insufficient contrast.
- The `#6D28D9` → `#4C1D95` gradient (145°) is the canonical app icon tile ground.
